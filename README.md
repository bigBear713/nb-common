## Installation
```bash
$ npm i @bigbear713/nb-common
// or
$ yarn add @bigbear713/nb-common
```

## 启动demo项目
- 安装依赖：
```bash
npm i
```

- 编译ng-common库
```bash
npm run build:lib
```

- 启动demo项目
```bash
npm start
```

- 部署demo
```bash
npm run build
```
