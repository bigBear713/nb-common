## Installation
```bash
$ npm i @bigbear713/nb-common
// or
$ yarn add @bigbear713/nb-common
```

## Start the demo project
- Install the dependencies:
```bash
npm i
```

- Build the nb-common lib
```bash
npm run build:lib
```

- Start the demo
```bash
npm start
```

- build the demo
```bash
npm run build
```
