## Feature
- 支持组件的更新策略为`ChangeDetectionStrategy.OnPush`;
- 支持在`standalone component`中使用;
- 支持以`standalone component`的方式引入;

## Installation
```bash
$ npm i @bigbear713/nb-common
// or
$ yarn add @bigbear713/nb-common
```

---

## 启动demo项目
- 安装依赖：
```bash
npm i
```

- 编译nb-common库
```bash
npm run build:lib
```

- 运行nb-common单元测试
```bash
npm run test:lib
```

- 启动demo项目
```bash
npm start
```

- 部署demo
```bash
npm run build
```
